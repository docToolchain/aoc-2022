==  Day 4: Camp Cleanup  ==

This solution is written in Kotlin.

The original puzzle can be found at https://adventofcode.com/2022/day/4


=== First Star

How does it work: +

Inside function _RuRePartOne_ the puzzle input is read line by line. Each line is split into the ranges of the elfs and all complete overlaps are counted at input == 1
